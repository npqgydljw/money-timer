# 赚钱计时器 需求文档

## 项目概述
一个精美的赚钱计时器展示页面，通过模拟赚钱过程带来视觉和听觉上的满足感。

## 功能需求

### 1. 计时与金额计算
- 每分钟赚钱 0.5 元（默认日收入 6000 元 / 24小时 / 60分钟）
- 实时显示累计赚钱金额和经过时间（HH:MM:SS）
- 每赚够 **N 元**触发一次庆祝效果，N 可在配置中自定义（默认 10 元）

### 2. 庆祝效果
- **礼花**：在屏幕随机位置绽放心形+彩色粒子礼花
- **弹窗**：居中弹出金色提示框，显示内容与语音播报一致（一行显示）
- **语音播报**：播放自定义播音文案，默认"尊敬的老板，您已赚到 xx 元"

### 3. 底部控制按钮
| 按钮 | 功能 | 样式 |
|------|------|------|
| 开始 | 启动计时器 | 蓝色发光边框 |
| 暂停 | 暂停计时 | 橙色发光边框 |
| 重置 | 清空所有数据，从零开始 | 红色发光边框 |

### 4. 顶部状态栏
- **左上角**：实时显示当前日期时间（格式：YYYY年MM月DD日 星期X HH:MM:SS），无边框无填充
- **右上角**：️ 配置按钮，点击打开配置弹窗

### 5. 配置弹窗（标题：系统配置）

#### 5.1 标题文案
- 单行输入框，默认值"今日个人收益"
- 清空时自动还原为"今日个人收益"
- 保存后页面大标题和浏览器标签页同步更新
- 保存到 localStorage，刷新页面不丢失

#### 5.2 每天赚多少钱
- 数字输入框，默认值 6000 元
- 下方实时显示换算结果（如"约 4.17 元/分钟"）
- **保存行为**：
  - 仅改变后续赚钱速度
  - 先冻结当前已赚金额到 accumulatedMoney
  - 重置 elapsedTime 为 0
  - 累计金额和经过时间不会归零
  - 里程碑自动同步当前总额，避免金额突变时重复触发庆祝

#### 5.3 每赚多少元庆祝一次
- 数字输入框，只允许输入整数，默认值 10 元
- 保存到 localStorage，刷新页面不丢失
- 保存后按照新的里程碑金额触发庆祝效果

#### 5.4 播音文案
- 多行输入框，默认值"尊敬的老板，您已赚到{money}"
- 最多支持输入 50 字
- 实时显示剩余可输入字数
- 剩余 10 字以内时提示文字变红
- 清空时自动还原为默认值
- `{money}` 为系统固定变量，会自动替换为实际金额（如"10元"）
- 保存到 localStorage，刷新页面不丢失

#### 5.5 播放声音
- 下拉选择 + 试听按钮
- 支持 4 种声音：
  | 选项 | 说明 |
  |------|------|
  | 甜妹女声 | 复刻林志玲的声音（默认） |
  | 青春男声 | 复刻张凌赫的声音，年轻活力男声 |
  | 成熟女声 | 复刻高圆圆的声音 |
  | 成熟男声 | 复刻赵又廷的声音，低沉稳重男声 |

- 试听按钮播放当前配置的播音文案（替换为"一百元"）
- 播放中按钮变为"⏹ 停止"，可随时终止
- 选择保存到 localStorage，刷新页面不丢失
- 默认选择"甜妹女声"，页面加载时自动播放欢迎语音

### 6. 键盘快捷键
| 按键 | 功能 |
|------|------|
| 空格 | 开始 / 暂停 切换 |
| R | 重置计时器 |

## 视觉设计

### 整体风格
- 科技风格深色主题（深蓝黑色背景）
- 蓝色（#00d4ff）为主色调
- 科幻背景图（行星+未来城市+水面反射）
- Orbitron + Noto Sans SC + Share Tech Mono 字体组合

### 按钮样式
- 所有按钮圆角：10px
- 按钮高度：80px
- 按钮宽度：140px
- 图标在文字左边（横向排列）
- 开始按钮：蓝色发光（rgba(0, 180, 255, 0.1)）
- 暂停按钮：橙色发光（rgba(255, 180, 0, 0.1)）
- 重置按钮：红色发光（rgba(255, 60, 60, 0.1)）

### 框架样式
- title-frame（标题框架）：宽度 500px，八边形切割边框
- money-frame（金额框架）：宽度 500px，八边形切割边框
- ¥ 符号在金额数字左边（同一行显示）

### 图标
- 左上角 Logo：三叶草图标（绿色）
- 标题前图标：三叶草图标（蓝色）
- 副标题装饰：三叶草图标

### 背景
- 使用 bg.png 作为页面背景（科幻行星+未来城市夜景）
- 暗色叠加层：35% 透明度的深色遮罩
- 网格线叠加层：微弱的蓝色网格
- CRT 扫描线效果

### 动画效果
- 页面背景：浮动装饰粒子
- 金额边框：蓝色光带沿八边形边框顺时针持续流动（跑马灯效果，约 3 秒一圈）
- 金额数字：无脉冲动画，仅静态显示
- 按钮：悬停上浮效果
- 配置弹窗：缩放弹出动画
- 礼花：彩色粒子向外扩散消散

### 响应式布局
- 桌面端和移动端自适应
- 移动端顶部栏变为垂直堆叠布局

## 数据持久化
以下配置保存到 localStorage：
- `moneyTimerTitle`：标题文案
- `moneyTimerVoice`：声音类型
- `moneyTimerVoiceText`：播音文案
- `moneyTimerMilestone`：每赚多少元庆祝一次

## 技术实现
- 纯 HTML + CSS + JavaScript（无框架依赖）
- 使用 Web Speech Synthesis API 实现语音播报
- 使用 CSS 动画和 JavaScript 动态创建粒子实现礼花效果
- 使用 clip-path 实现八边形切割边框效果
- 使用 SVG stroke-dasharray 动画实现边框跑马灯光效
